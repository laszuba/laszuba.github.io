# {{ page.title }}

Spacejunk was my first ever robot, built with the generous help of my dad, which solidified my interest in robotics and engineering. It uses two power window motors for drive, and is built with a simple riveted aluminum chassis. The motor controller is a relay H-bridge. Two years after the initial construction, it was converted for autonomous control using the electronics from a Solarbotics Sumovore.
